 ## Conjunto de dados
Análise de dados sobre diversidade na área de tecnologia no ano de 2022. Os dados foram coletados na pesquisa anual do Datahackers de 2022. Na análise utilizamos Python com as bibliotecas Pandas, Numpy e matplotlib para realizar análise exploratória e identificar os a distribuição de profissionais de tecnologia e a diversidade na área. Além da análise realizada em python utilizando as bibliotecas pandas, matplotlib e numpy realizamos a visualizações mais completas dos dados no looker studio.
#### Visualização de dados no Looker Studio : https://lookerstudio.google.com/reporting/1cd27196-030b-44c9-b4d2-d7913149bd83

![Dist_genero](https://github.com/JNarimatsu/assets/raw/main/pm_pessoas_genero.png)

Conseguimos verificar que em nossa base de dados temos um número maior de pessoas que se identificaram com o gênero masculino.
